### 🚀 Sobre el Proyecto
Bienvenidos a la experiencia interactiva oficial de **LeonelB2840 OFFICIAL 2.0**. 
Este no es solo un juego, es la representación de nuestra lucha por mantener la tecnología libre. Ayuda al Pingüino 🐧 a recolectar los archivos esenciales para la **Store 2.0** mientras esquivas los bloqueos de Copyright 🚫.

### 🕹️ Cómo Jugar
1. **Objetivo:** Recolecta la mayor cantidad de paquetes 📦 para subir el nivel de la Store.
2. **Controles:** - **PC:** Mueve el mouse para desplazar al pingüino.
   - **Móvil:** Desliza el dedo por la pantalla.
3. **Modo Turbo ⚡:** Mantén presionado el botón rojo para activar la ralentización de tiempo (Matrix Mode) y esquivar obstáculos difíciles.

### 🛠️ Características Técnicas
- **Motor:** HTML5 / JavaScript Puro (Sin frameworks externos).
- **Audio:** Sintetizador de ondas en tiempo real.
- **Optimización:** Ejecución fluida en dispositivos de gama baja (Mentalidad de Rico).
